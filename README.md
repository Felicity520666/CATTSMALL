# CATTSMALL
## About This Project
Hi everyone! This is a small and simple project that I created to practice my html and css skills! At the same time, I wanted to shout out to Catt Small! I learnt about her and was thankful for her contributions to making women in tech and games more visible. So this project was created to honor her and improve my skills of making websites.

## Features
- Simple, cute, and clean design
- Includes an image of Catt Small
- Catt Small's biography and introduction
- A link to her games if you want to check them out!
